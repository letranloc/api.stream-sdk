Scene composition: WebRTC

This example project enables you to see and test how to interact with scene composition while bringing in WebRTC guests

To get started enter your API Key and an RTMP URL and key to output to. When ready hit start broadcast to start the broadcast

While viewing the destination stream, click "Join WebRTC" to connect your local session to WebRTC, forwarding webcam and microphone.

Ensure to click "Stop broadcast" once you are done testing.
